# EZENO-DENTIFRICE
Vente de dentifrice


EZENO - Site de Présentation

Ce projet est une vitrine web conçue pour présenter la gamme de dentifrices aux huiles essentielles EZENO. Le site met en avant les produits et facilite la mise en relation directe avec les clients via WhatsApp.

Fonctionnalités 
Présentation des produits : Affichage visuel des différentes gammes.  
Interface responsive : Adaptée pour une consultation sur mobile et ordinateur.  
Intégration WhatsApp : Boutons d'action rapide permettant aux clients de demander des prix ou de passer commande directement par message.  

Technologies utilisées
HTML5 : Structure du contenu.  
CSS3 : Design, mise en page en grille (grid) et responsivité.  

Structure du projetindex.
html : Fichier principal contenant la structure du site.  
style.css : Feuille de style gérant l'apparence et le responsive design.  
/images : Dossier contenant les visuels des produits.
